# Notas
```
1. Conectar su base de datos en postgres o en cualquier gestor que manejen.
```
```
2. Ejecutar las respectivas migraciones:
➡️ python3 manage.py makemigrations
➡️ python3 manage.py migrate
```
```
2. Crear super usuario:
➡️ python3 manage.py createsuperuser
```
```
2. Ejecutar el servidor:
➡️ python3 manage.py runserver
```
